This guide provides the syntax for the most commonly used and important HTML5 tags, categorized by their function.

Essential HTML Tags

1) <!DOCTYPE html> - Declares the document to be HTML5.

Example: <!DOCTYPE html>

2) <html> - Root element of an HTML page.

Example: <html lang="en">...</html>

3) <head> - Contains meta-information about the document.

Example: <head>...<title>Page Title</title>...</head>

4) <title> - Sets the title of the document.

Example: <title>My Awesome HTML Page</title>

Output: My Awesome HTML Page

5) <body> - Contains the visible content of the HTML

Example: <body><h1>Hello World</h1></body>

Output: "Hello World"

Text Content & Headings

1) <h1> to <h6> - Define HTML headings

Example: <h2>Section Title</h2>

Output:

Section Title

2) <hr> - Creates a thematic break

Example: <p>Content Above</p> <hr> <p>Content Below</p>

Output:

Content Above

Content Below

3) <blockquote> - Quotation block

Example: <blockquote>A single quote.</blockquote>

A single quote.

4) <pre> - Preserves formatting

Example: <pre>Line 1
Line 2</pre>

Line 1
Line 2

5) <p> - Paragraph tag

Example: <p>This is a paragraph.</p>

This is a paragraph of text.

Lists

1) <ul> - Unordered list

Example: <ul><li>Coffee</li></ul>

  • Coffee

2) <ol> - Ordered list

Example: <ol><li>First</li></ol>

  1. First

3) <dt>, <dd>, <dl> - Description list

Example: <dl><dt>HTML</dt><dd>Language</dd></dl>

HTML
Language

Forms & User Input

1) <form> - Defines an HTML form for user input.

Example: <form action="/submit">...</form>

2) <input> - Defines an input field.

Example: <input type="text" placeholder="Your Name">

3) <label> - Defines a label for an <input> element.

Example: <label for="name">Name:</label>

4) <textarea> - Defines a multi-line text input control.

Example: <textarea rows="3" cols="20">...</textarea>

5) <select> and <option> - Creates a dropdown list.

Example: <select><option>A</option><option>B</option></select>

6) <button> - Defines a clickable button.

Example: <button type="submit">Send</button>

Tables

1) <table> structure (<table>, <tr>, <th>, <td>, <caption>)

Example: <table border="1"><caption>Data</caption><tr><th>Header</th><td>Value</td></tr></table>

2) <thead>, <tbody>, <tfoot> - Table Structure Groups.

Example: <table> <thead>...</thead> <tbody>...</tbody> <tfoot>...</tfoot> </table>
